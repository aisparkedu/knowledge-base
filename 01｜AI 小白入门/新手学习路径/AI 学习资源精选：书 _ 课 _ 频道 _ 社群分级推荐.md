# AI 学习资源精选：书 / 课 / 频道 / 社群分级推荐

> 📚
> **这一篇是 AI 学习资源整理。读完你能：**
> - 知道入门 / 进阶 / 高阶各自该看什么
> - 看到一个新的"AI 资源"知道值不值得花时间
> - 识别"伪干货"和"真深度"
> - 有一份按阶段叠加的资源叠加表

## 1. 资源的 4 种类型 + 适用阶段

| **类型** | **适合阶段** | **优点** | **缺点** |
|-|-|-|-|
| 书籍 | 认知建立 | 体系完整、不浮夸 | 更新慢、太厚容易放弃 |
| 视频课程 | 工具上手 + 实战 | 直观、跟着做 | 带货 / 灌水风险高 |
| YouTube / B 站频道 | 持续跟进新动态 | 新鲜、轻量 | 追新焦虑、碎片化 |
| 社群 / 论坛 | 实战 + 答疑 | 解决具体问题、有同伴 | 信息密度低、容易刷 |

## 2. 入门书籍 5 本

- **《这就是 ChatGPT》** 沃尔夫勒姆——用最少的数学讲清原理，新手第一本
- **《Hands-On Large Language Models》** Jay Alammar——动手做 LLM 应用，进阶必读
- **《LLM Prompt Engineering for Developers》**——Prompt 工程深挖
- **《Generative AI with LangChain》**——RAG / Agent 实战入门
- **《Designing Machine Learning Systems》** Chip Huyen——AI 系统设计思维

## 6. 怎么识别"伪干货"

> 🚨
> **5 个伪干货标志：**
> 1. 标题党：「99% 的人不知道的 XX 秘密」
> 2. 清单堆砌：「100 个必备 AI 工具」（看完什么也记不住）
> 3. 无证据：没具体例子、没数据、没复现路径
> 4. 追新焦虑：「不会用就被淘汰」（卖焦虑卖课）
> 5. 无实战：作者自己看起来从来没真用过

## 7. 推荐的学习节奏

- **第 1 个月：**1 本书 + 1 个系统视频课，把基础打牢
- **第 2-3 个月：**每天逛 1 个频道，跟实时进展
- **第 4-6 个月：**开始加入 1-2 个社群，把同伴效应用起来
- **半年以上：**开始读论文 / 关注研究类社群
