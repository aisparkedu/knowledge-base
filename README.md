## 📖 这是什么

**AI Spark 知识库**是一个由**一线在做 AI 业务、跑自媒体账号的实战者**共同维护的开源知识库。

我们写的，都是当下真在用的——而不是二手知识。目标只有一个：

> **让每个想用 AI 的人，都能从「看不懂」走到「跑通第一个工作流」，再走到「靠 AI 提效增收」。**

普通人不用再花 999 买课，就能在这里拿到能直接上手的工具、Prompt、工作流和真实案例。

### 内容来自两个源头

- **我们维护的实战教程**——团队按各自专长持续输出，每周更新
- **全网爆款文章的沉淀**——在 X / 公众号 / 抖音等平台跑出过 10w+、100w+ 的内容，整理归档进来

### 现在普通人能用 AI 做什么

- 用智能体把重复劳动自动化，每天省 2-4 小时
- 用 Vibe Coding 让没有编程背景的人，也能搭出自己的工具、小型 SaaS、自动化系统
- 用 AI 找选题、写稿、改稿、生成爆款封面，跑通短视频脚本到口播的全流程
- 用 AI 把「内容 - 引流 - 转化」形成闭环
