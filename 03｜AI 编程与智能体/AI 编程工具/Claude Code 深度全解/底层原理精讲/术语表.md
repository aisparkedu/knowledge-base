# 术语表

本指南中你将遇到的术语的字母顺序参考。涵盖 Claude Code 特有概念、社区创造的模式以及 AI 工程词汇。标准 CS/DevOps 术语（JWT、CI/CD、REST）不在此列——请自行查阅。

**格式**：术语 | 定义 | 分类 | 子分类
